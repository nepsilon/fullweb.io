TIP OF THE WEEK

Not using versioning on your configuration files and editing them with Vim?
Use Vim’s backup option to automatically keep an archive of past versions:
In your ~/.vimrc:
"Turn on backup option set backup "Where to store backups set backupdir=~/.vim/backup// "Make backup before overwriting set writebackup "Overwrite the original file set backupcopy=yes "Meaningful backup name au BufWritePre * let &bex = '@' . strftime("%F.%H:%M")
