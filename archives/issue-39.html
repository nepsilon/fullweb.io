TIP OF THE WEEK

Display a custom message when you log in with SSH

SSH gives us a way to display a custom welcome message when a user log in. You can make this message dynamic thanks to update-motd, but here we’ll talk about the SSH built-in feature.
Here is how to do in 3 steps:

# 1. Put your message in /etc/issue.net $ vim /etc/issue.net # (Check out cowsay to have a cow tell your msg)
# 2. Turn on this SSH feature $ vim /etc/ssh/sshd_config
...
# Uncomment this line and save+exit: #Banner /etc/issue.net # Just remove the first # character to uncomment
# 3. Restart SSH: $ sudo etc/init.d/sshd restart # Now whenever login in, the message will appear :)
