TIP OF THE WEEK
 

Use $$() not $() to return an array of elements

A tip I read on Remy Sharp’s blog that I wanted to share with you all:

The Devtools console gives you access to the $() function (even if you don’t include jQuery/Zepto, etc). It’t nice but doesn’t help if you need to return a collection of elements.

That’s where you can use the $$() function, also provided by the console. And the best part is that it doesn't return a NodeList but a real Array.
So you can use .map, .forEach directly on it 👏.
