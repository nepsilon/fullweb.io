TIP OF THE WEEK

Let’s talk about jq, the command line JSON processor.
JQ works the UNIX way, and can use it the same way you use sed, grep and the likes.
JQ is very flexible and supports a lot of options, be sure to check it’s documentation.
Here are 2 simple examples to give you an idea:

# Piping JSON to JQ # '.' denotes the root of the JSON # Return the first element of the root array ... | jq '.[0]'

# Now only keep some keys: # id and message.author ... | jq '.[0] | {id, message.author}'
